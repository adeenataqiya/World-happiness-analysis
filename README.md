# World-happiness-analysis
World Happiness Report 2018 - EDA Project
Objective
Analyze the World Happiness Report 2018 dataset and identify factors affecting happiness.

Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Dataset
World Happiness Report 2018

Visualizations
Histogram
Boxplot
Scatter Plot
piechart
line plot 6.Heatmap
Key Insights
1.Finland is the happiest country in the dataset.

2.Countries with higher GDP generally have higher happiness scores.

3.Healthy life expectancy is positively related to happiness.

4.Social support has a strong relationship with happiness.

5.Most countries have happiness scores between 4 and 7.
